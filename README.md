# BookTracker
**Автор:** Шамин Кирилл

### Описание
BookTracker - это графическое приложение на Python для ведения списка прочитанных книг. 
Оно позволяет сохранять данные в файл, чтобы они не терялись после закрытия программы.

### Основные функции
1. **Добавление книг**: Укажите название, автора, жанр и количество страниц.
2. **Валидация**: Программа проверяет, чтобы поля не были пустыми, а страницы были числом.
3. **Фильтрация**: 
   - Можно искать книги по конкретному жанру.
   - Можно искать книги с количеством страниц больше указанного числа.
4. **Хранение данных**: Все книги автоматически сохраняются в файл `books.json`.

### Инструкция по запуску
1. Установите Python (версия 3.x).
2. Скачайте файл с кодом (например, `main.py`).
3. Запустите программу командой: `python main.py`.

### Пример использования
- Введите "Гарри Поттер", "Дж. Роулинг", "Фэнтези", "500".
- Нажмите "Добавить книгу".
- В фильтре "Минимум страниц" введите "400" и нажмите "Отфильтровать". Вы увидите свою книгу в списке.
